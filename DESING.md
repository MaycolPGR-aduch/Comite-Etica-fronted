# DESIGN.md

## Sistema de diseño del frontend

Este documento define las reglas visuales para el frontend del Sistema Web para la Gestión y Evaluación de Protocolos del Comité de Ética.

La interfaz debe ser institucional, moderna, minimalista, profesional, clara y fácil de usar.

---

## Personalidad visual

La interfaz debe transmitir:

- Confianza
- Orden
- Formalidad
- Transparencia
- Trazabilidad
- Profesionalismo
- Claridad administrativa

No debe verse como una red social, landing comercial o aplicación demasiado informal.
