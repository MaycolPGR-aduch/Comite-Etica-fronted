# CLAUDE.md

## Proyecto

Frontend del Sistema Web para la Gestión y Evaluación de Protocolos del Comité de Ética.

Stack:
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React
- FastAPI como backend

## Reglas obligatorias

- No instalar paquetes nuevos.
- No cambiar rutas existentes.
- No romper lógica funcional.
- No modificar servicios, mocks ni tipos salvo necesidad justificada.
- No rediseñar pantallas de forma aislada.
- Priorizar componentes globales.
- Leer DESIGN.md antes de hacer cambios visuales.
- Trabajar por fases.
- Antes de modificar código, presentar plan.
